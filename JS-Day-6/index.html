Types of function:

  • Regular function
  • Arrow function
  • Callback function
  • First class function
  • High order function : map,reduce,filter
  • Anonymous function
  • Self invoking function or IIFE(imediately invoked function expression): To obtain data privacy and don't give access to call anywhere we use IIFE.

First class function:

  • We can treat a variable as a function
  • We can pass a function inside another function as an argument
  • We can return a function from another function

Note--Return: it will return only once bydefault it acts like break statement and once the code hit the return statement it will terminate the loop or move out of the block
